# 🐦 Twitter Clone - Tailwind CSS

A pixel-perfect, responsive Twitter/X.com UI clone built with **Tailwind CSS** to demonstrate modern utility-first styling and clean component architecture.

## 🎯 Project Overview

This project is a **learning-focused clone** of Twitter/X.com's user interface, built entirely with **Tailwind CSS utility classes**. The goal is to master responsive design, modern CSS frameworks, and component-based styling without writing custom CSS.

---

## ✨ Features

- 🎨 **Utility-First Styling**: 100% Tailwind CSS — no custom CSS written
- 📱 **Fully Responsive**: Mobile-first design that adapts seamlessly to all screen sizes
- ⚡ **Fast Development**: Vite dev server for instant hot reload
- 🧩 **Component Architecture**: Reusable, modular UI components
- 🎭 **Pixel-Perfect UI**: Accurate recreation of Twitter's design language
- 🌈 **Modern Design Patterns**: Flexbox, Grid, and spacing utilities

## 🚀 Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/mayurdhanurkar/Twitter-Clone-Tailwind.git
   cd Twitter-Clone-Tailwind
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start development server**
   ```bash
   npm run dev
   ```

4. **Build Tailwind CSS** (in a separate terminal)
   ```bash
   npm run build
   ```

5. Open your browser and navigate to `http://localhost:5173`

## 🎨 Tailwind Features Used

- **Layout**: Flexbox (`flex`, `justify-center`, `items-center`), Grid (`grid`, `grid-cols-*`)
- **Spacing**: Margin (`m-*`), Padding (`p-*`), Gap (`gap-*`)
- **Typography**: Font sizes (`text-*`), weights (`font-bold`), colors (`text-gray-*`)
- **Responsive**: Breakpoints (`sm:`, `md:`, `lg:`, `xl:`)
- **Interactivity**: Hover states (`hover:`), focus states (`focus:`)
- **Colors**: Custom color palette and opacity utilities
